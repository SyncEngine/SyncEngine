# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability in **SyncEngine**, please report it directly to **info@syncengine.io**.

- Reports will be handled **personally and promptly** by the maintainers.
- We will keep you updated on the progress and resolution of the issue.

---
## Current Status
SyncEngine is currently in **beta phase** and operates as a **rolling release**. Security updates and fixes will be applied continuously until the first stable (non-beta) release.
