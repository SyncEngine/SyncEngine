
# Call examples

## Step config
<p>
To add a number to your price use the following step config: <br>
	{
		"plus": 20,
		"module": "EditPrice"
	}
</p>
<p>
To capitalize each word of your specified tag:<br>
	{
		"tag": "title",
		"module": "Capitalization"
	}
</p>

<p>
For these examples your call can look as the following: <br>
	{
		"price": 12,
		"title" : "shirt with sleeves",
		"language": "NL"
	}
</p>
